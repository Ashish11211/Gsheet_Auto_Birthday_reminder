# 🎂 Birthday Reminder Automation (Google Apps Script)

## 📌 Description
Automated Birthday Reminder system using Google Apps Script integrated with Google Sheets. It sends reminder notifications and personalized birthday wishes automatically.

## 🚀 Features
- Auto birthday detection from Google Sheet
- Sends reminder to admin
- Sends birthday wishes to user
- Fully automated (trigger-based)

## 🛠 Tech Stack
- Google Apps Script
- Google Sheets

## ⚙️ How It Works
1. Store birthday data in Google Sheets
2. Script checks daily using trigger
3. Sends reminder & birthday message automatically
